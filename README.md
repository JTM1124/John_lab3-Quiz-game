# Quiz Game

## Description
A simple C++ console program that allows the user to play a short multiple choice quiz based on general knowledge.

## How to Run
1. Open the project folder in Visual Studio.
2. Press Ctrl+F5 to build and run.
3. Answer five questions with A B C or D.

## Sample Input / Output
Input:  A  B  C  A  D
Output: Total Score: 3 / 5
